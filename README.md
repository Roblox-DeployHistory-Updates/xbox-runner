# xbox-runner

Fetches Roblox Xbox deployments, includes a [web viewer](https://roblox-deployhistory-updates.github.io/xbox-runner/) that displays history with metadata like version names, release dates, and download links.

The Google sheets which contains versions also prior to the tracker can be found [here](https://docs.google.com/spreadsheets/d/1QB6SexYxoWUr2LUEaZJcmMf6eBOFAb3sozSWXSvHU5U/).
